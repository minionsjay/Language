# 1994年广播法
## 有害内容类别指南

1. 本套指南针对《网络安全行为准则》，为所有最终用户提供了说明性但并非详尽无遗的有害内容示例，并为儿童提供了不适宜内容示例。

### 有害/不适宜内容示例¹

| 类别 | 所有最终用户 | 儿童 |
|------|--------------|------|
| **性内容** | • 描绘露骨性行为的内容。<br>• 描绘或宣扬异常性行为的内容，例如：<br>  ◦ 乱伦。<br>  ◦ 兽交。<br>  ◦ 恋尸癖。<br>  ◦ 恋童癖。<br>• 涉及或鼓励《刑法典》、《儿童与青少年法》及《妇女宪章》规定的性犯罪的内容，例如：<br>  ◦ 传播儿童性虐待材料。<br>  ◦ 未经同意传播偷窥或私密图像或录音。<br>  ◦ 与未成年人进行性交流。<br>  ◦ 鼓励性侵犯的内容。 | • 描绘性活动的内容，即使是在虚构情境中。<br>• 暗示或模糊描绘性活动的内容。<br>• 包含性语境下裸体的内容。<br>• 包含频繁使用旨在获得性满足的性暗示或影射的内容。 |
| **暴力内容** | • 描绘对人员施加极端暴力或残忍行为的内容，例如肢解、斩首。<br>• 暗示/模糊描绘暴力或血腥的内容，例如经过模糊或遮挡处理的暴力内容。<br>• 描绘对人或动物进行虐待或折磨的内容，例如展示造成严重伤害和痛苦的近距离细节的片段。<br>• 提供有关恐怖主义或极端暴力行为指导或鼓励此类行为的内容。<br>• 与恐怖主义或内部安全相关的犯罪（例如，教唆或合谋资助或为实施恐怖主义行为提供物质支持，以及组织、训练或装备被禁止的准军事组织）的内容。<br>• 涉及煽动暴力、大规模骚乱或暴动的内容，无论是一般性的还是针对特定特征人群的。 | • 暗示/模糊描绘对人或动物进行虐待或折磨的内容，例如显示部分遮挡的虐待行为的内容。 |
| **自杀与自残内容** | • 鼓励、认可或提供自残（包括可能导致死亡的行为）或自杀指导的内容。<br>• 描绘自残图形细节（例如，展示伤口或伤害）的内容。<br>• 涉及教唆自杀的内容。 | • 暗示性或非细节性描绘自残的内容（例如，愈合的疤痕、模糊的视觉图像）。<br>• 可能鼓励或误导儿童从事可能导致身体伤害的危险行为，或提供相关指导的内容，例如涉及危险特技或元素（如火、窒息、尖锐物体、高处）的挑战。<br>• 可能鼓励、认可或指导儿童如何故意伤害自己身体，从而对健康产生负面或不利影响的内容（例如，宣扬饮食失调的内容）。 |
| **网络霸凌内容** | • 可能对目标个人或群体造成骚扰、恐慌或困扰的内容，例如：<br>  ◦ 使用威胁性、辱骂性或侮辱性的言辞或行为。<br>  ◦ 在线发布或威胁发布目标个人的个人信息，或可能导致目标个人被识别的他人信息。<br>  ◦ 与目标个人特征相关的威胁、贬低性或非人性化的言论或提及。<br>  ◦ 对目标个人进行非法跟踪。<br>  ◦ 冒充目标个人或群体，以其名义使用威胁性、辱骂性或侮辱性的言辞。 | • 可能嘲弄、羞辱或使目标个人感到尴尬的内容，例如：<br>  ◦ 将儿童置于尴尬情境中的编辑内容，并配以文字进行嘲弄和讥笑。<br>  ◦ 基于智力、行为或身体特征等属性对儿童进行的负面陈述或提及。<br>• 鼓励或呼吁对其他儿童进行协调性骚扰/霸凌的内容。 |
| **危害公共健康的内容** | • 为未经新加坡相关当局批准的目的，推广可能对个人造成严重伤害或死亡的管制医疗或保健产品的内容。<br>• 涉及可能阻碍、规避或违反为保护公共健康而设立的措施的行为的内容。 | |
| **助长恶习与有组织犯罪的内容** | • 涉及卖淫、非法赌博、非法放贷、人口贩卖、欺诈、诈骗和勒索的内容。<br>• 涉及精神药物、危险毒品、麻醉品或烟草产品相关犯罪的内容，包括导致年轻人或弱势群体贩卖或吸食毒品，或安排或策划聚会以吸食或贩卖管制毒品的内容。 | |

¹ 在确定内容是否为有害内容时，内容在文学、艺术、社会、文化、教育、科学或其他方面的价值、重要性或意义将是相关考量因素。
